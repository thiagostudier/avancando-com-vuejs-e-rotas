# Escopo e objetivo

Fundamentos da Seção 8 do curso "Curso Vue Js do zero ao avançado na prática" (Udemy)

(a) VueJs

## Fundamentos aplicados

## Iniciando...

- `cd avancando-com-vuejs-e-rotas`

> _Parte da configuração: Necessário instalar o yarn

- `yarn install`
